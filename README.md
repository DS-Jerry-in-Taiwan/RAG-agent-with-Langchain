### 專案核心
建構 **PDF 智能搜尋引擎**，整合 Document Loader、Embedding、Vector Store 三層技術棧，實現語意檢索與 RAG 系統落地

### 技術架構

**文件向量化管線**
-  運用 Document Loader 解析 PDF 並切分段落，建立結構化預處理流程
-  採用 Embedding 將文本轉換為高維度向量，保留語意資訊與上下文關聯性

**向量檢索系統**
-  建構 Vector Store 管理文件向量，支援高效相似度檢索
-  查詢向量化後透過相似度計算快速定位相關段落，優於傳統關鍵字比對

**RAG 整合應用**
-  整合檢索引擎與 LLM 工作流，將檢索段落作為上下文輸入
-  建構檢索增強生成架構，實現知識庫問答系統端到端解決方案

### 應用場景
適用於企業知識庫智能問答、技術文件檢索、法規條文查詢等精準語意檢索場景，支援大規模非結構化文件智能化管理
